## 🌟 Features

- **Native i18n Support**: Full support for English and Spanish using Astro's native internationalization.
- **AI-Driven Engineering**: Showcasing expertise in modern AI tools (Claude Code, Gemini CLI, MCP).
- **Functional Programming Stack**: Built with a focus on the Scala ecosystem (Http4s, Smithy, Cats).
- **Command Palette**: Keyboard-accessible navigation with `Cmd + K`.
- **Minimalist Design**: Clean, fast, and optimized for performance.

## 🛠️ Tech Stack

- [**Astro**](https://astro.build/) - Modern static site generator.
- [**Typescript**](https://www.typescriptlang.org/) - Type-safe development.
- [**HotKeyPad**](https://github.com/ssleptsov/hotkeypad) - Keyboard shortcuts management.
- [**Tailwind CSS**](https://tailwindcss.com/) - Utility-first styling.

## 🚀 Getting Started

### 1. Installation

I use [pnpm](https://pnpm.io/) as the dependency manager.

```bash
# Install dependencies
pnpm install
```

### 2. Configuration

Content is managed through JSON files:
- `cv_en.json`: English version.
- `cv_es.json`: Spanish version.

### 3. Development

```bash
# Run development server
pnpm dev
```

1. Open [**http://localhost:4321**](http://localhost:4321/) to see the results 🚀

## 🧞 Commands

| Icon | Command | Action |
